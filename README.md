# Nacklace of Water Repulling

Minecraft MOD


## 追加するアイテム

### 水除けの首飾り

#### 機能

所持しているプレイヤーの周りの水を消し去ります。

アイテムを右クリックでオンオフを切り替えられます。

#### レシピ

ネザースター + バケツ
